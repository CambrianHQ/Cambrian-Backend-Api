# Bruno Starter Collection

This folder contains Git-friendly Bruno examples for common Cambrian manual QA flows.

Included examples:

- `auth/Login.bru`
- `catalog/Get Catalog.bru`
- `library/Get Library.bru`
- `health/Health Check.bru`
- `environments/staging.bru`

These files are intentionally lightweight starter requests rather than a full mirrored collection.
